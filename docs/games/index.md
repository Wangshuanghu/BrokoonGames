## 如何在集宣组织宣传网站中添加自己的服务器展示位置？

### 一、 提供服务器的必要信息，包括：

  1. 服务器名称
  2. 服务器图标（可选）
  3. 服务器信息
  4. 服务器描述
  5. 服务器版本
  6. 宣传链接（可以是QQ群的入群链接、服务器的宣传网站链接，或者宣传视频的链接）

### 二、 向负责人或已经加入GitHub编辑的成员发送包含上述服务器必要信息的申请。

``` txt
服务器名称：
服务器图标：
服务器信息：
服务器描述：
服务器版本：
宣传链接：
```
